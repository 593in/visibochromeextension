Como dijo Lifehacker.com -- "Gran administrador de pestañas"!

Cuando trabajas con varias pestañas al mismo tiempo, es difícil recordar cuál es la próxima a la que quieres ir. TabJump destaca pestañas usadas frecuentemente, así como también pestañas relacionadas, para que puedas saltar fácilmente entre ellas. ¡Y nunca volverás a preocuparte por cerrar pestañas por accidente, ya que TabJump puede proteger e incluso restaurar las pestañas cerradas!

Características 
- Saltar a pestañas usadas frecuentemente
- Saltar a pestañas relacionadas (pestañas del mismo sitio, etc) 
- Restaurar pestañas cerradas recientemente
- Bloquea una pestaña para evitar el cierre accidental (sólo funciona para pestañas abiertas luego de la instalación de TabJump) 

Descargo de responsabilidad 
- Debido a las limitaciones de Chrome, las extensiones no pueden restaurar el historial de las pestañas cerradas. Si necesitas el historial de una pestaña cerrada hace poco, puedes pulsar Ctrl + Shift + T para abrir una por una.
