Gérez au mieux vos sessions de navigation facilement avec FreshStart! ! Si vous avez des groupes de sites web importants au travail comme à la maison, ou si vous avez plusieurs utilisateurs sur votre Google Chrome, vous devriez essayer cette extension!

New in 1.4
- Session Import and Export! Export your sessions as plain text, send it to a friend or import it to another machine
- Added confirmation dialogs when deleting sessions

Fonctionnalités
- Sauvegardez tous vos onglets ouverts dans une nouvelle session, ou sélectionnez à la main ceux que vous voulez
- Les sessions sont enregistrées dans le dossier FreshStart des favoris, vous permettant de les synchroniser avec d'autres ordinateurs
- Récupérateur de crash, sauvegarde automatique de vos fenêtres
- Restaurez une session dans une nouvelle fenêtre, ou fusionnez là dans la fenêtre courante en maintenant CTRL-Click (Windows, Linux) ou CMD-Click (Mac).


Prochaines Fonctionnalités
- Meilleure interface d'édition de session
