Gestisci la tua sessione di navigazione facilmente con FreshStart! Se si dispone di diversi insiemi di siti web che ti interessano al lavoro, a casa o in momenti diversi, o se si dispone di più utenti sul tuo Chrome, si dovrebbe dare a questo una prova!

New in 1.3
- Editing di seduta è qui! Facilmente si fondono nuove schede in una sessione esistente o rimuovere le schede indesiderati
- Aggiunto un pulsante per selezionare tutti i / le schede nessuno da uno scatto

Caratteristiche
- Salvare tutte le schede aperte per una nuova sessione, a mano o scegliere cosa essere salvato
- Le sessioni vengono salvati nel "FreshStart Sessions" nella cartella dei preferiti, consente di sincronizzare con altri computer
- Recupero di Crash - autosave le finestre ogni pochi minuti
- Ripristino configurazione di una sessione in una nuova finestra, o si fondono con la finestra corrente da ctrl-click (Windows, Linux) o cmd-click (Mac).

Prossimi Caratteristiche
- Meglio sessione di modifica di dialogo
