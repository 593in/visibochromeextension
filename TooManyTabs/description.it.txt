Se avete già provato ad avere pù di 20 schede Chrome aperte contemporaneamente, vi siete resi conto che non si vede nemmeno l'icona della scheda, e quindi nemmeno il titolo della scheda stessa.
L'Estensione Troppe Schede Chrome serve proprio per gestire meglio questo elevato numero di schede ed è stato sviluppato dagli stessi creatori di TooManyTabs per Firefox

Novità nella versione 1.4
- Visualizza le schede di tutte le finestre
- Scelte rapide da tastiera personalizzabili

Caratteristiche
- Hai una vista panoramica di tutte le schede aperte
- Hai una anteprima chiara del contenuto di ogni scheda che si aggiorna mentre si usa l'estensione
- Ricerchi istantaneamente le schede
- Ordini le schede per data di creazione, per dominio o per titolo
- Ripristini le schede chiuse di recente
- Sospendi le schede inutilizzate per salvare memoria,
- Organizza le schede con colonne personalizzate
- Personalizza i temi
- Puoi ridimensionare la finestra
- Puoi scegliere se mostrare il numero di schede sull'icona presente sulla barra degli strumenti Chrome

Caratteristiche in corso di sviluppo
- Ricerca delle schede sospese
- Miglioramenti nelle colonne sospese

Attenzione

Per caratteristiche proprie di Chrome, le estensioni non possono ripristinare la cronologia delle schede chiuse.
Se vuoi riaprire una scheda chiusa di recente, puoi premere i tasti Ctrl+Shift+T e riaprire le schede chiuse una per una.
