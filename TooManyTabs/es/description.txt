¿Alguna vez has intentado tener 20 o más pestañas abiertas en Chrome? Ni siquiera se puede ver el icono de las pestañas, por no mencionar sus títulos! Si eso te sucede, necesitas TooManyTabs, la extensión que gestiona las pestañas, reduce el desbordamiento de pestañas, y guarda tu cordura!

De los creadores de TooManyTabs para Firefox, TMT ha llegado a Chrome para ayudarte con tus pestañas!

Características
- Importa los datos de las pestañas desde Firefox (Nuevo)
- Vista de pájaro de todas las pestañas abiertas
- Vista previa clara del contenido de cada pestaña (se llena gradualmente a medida que se utiliza)
- Búsqueda de pestañas instantánea
- Ordena las pestañas por fecha de creación, dominio o título
- Restaura pestañas cerradas recientemente
- Suspende pestañas inactivas para guardar memoria, organizadas con columnas personalizadas
- Temas personalizados

Próximas características
- Búsqueda de pestañas supendidas
- Mejoras en la columna  de pestañas suspendidas

Descargo de responsabilidad
- Debido a las limitaciones de Chrome, las extensiones no puede restaurar el historial de las pestañas cerradas. Si usted necesita de el historial de una pestaña cerrada hace poco, puede pulsar Ctrl+Shift+T para abrirlas una por una.
