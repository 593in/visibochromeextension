Incredible StartPage ist ein produktiver Startseiten-Esatz für Chrome. Schenken Sie Ihren Lesezeichen einen wunderschönen Rahmen, und genießen Sie unsere praktischen Funktionen!

Wie auf DownloadSquad gesehen: http://goo.gl/iyYe

New in 0.9
- Notepad sync
- Post notes to Google App accounts
- More theme options, including using custom gradient as background
- Bundled default wallpaper for faster loading
- Recently closed tabs options

Funktionen:
- anpassbares, farbiges Design
- einfacher Zugriff auf Ihre Lesezeichen, welche sich in einem großen Rahmen befinden.
- leichtes Wiederherstellen von geschlossenen Tabs
- Machen Sie Notizen und schicken Sie Diese mit einem Klick zu Gmail.

Schauen Sie sich ein vollständiges Tutorial auf unserem Blog an: http://goo.gl/zHxq

Disclaimer
- Aufgrund von Chrome Beschränkungen können Erweiterungen nicht den Verlauf von geschlossenen Tabs wiederherstellen. Falls Sie den Verlauf von einem kürzlich geschlossenem Tab benötigen, können Sie mit Strg+Shift+T diesen Verlauf schrittweise wiederherstellen.
- Zur Information: Um die original Chrome Startseite wiederherzustellen, gehen Sie zu "Erweiterungen" von dem "Schraubenschlüssel" Menü, und deaktivieren/deinstallieren die Erweiterung.
